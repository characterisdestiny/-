##  SecureFileGuard

> **실시간 악성 코드 분석 기반 보안 파일 전송 시스템**  
> Java Socket 기반으로 클라이언트가 전송한 파일을 서버에서 자동 분석 및 차단하는 보안 시뮬레이션 프로젝트입니다.

---

##  프로젝트 개요

SecureFileGuard는 단순한 파일 전송이 아닌,  
전송된 파일을 서버에서 **보안 기준에 따라 자동 분석**한 뒤  
 정상 파일은 저장,  
 악성 의심 파일은 즉시 차단 및 삭제하는 **보안 응용 프로그램**입니다.

---

##  주요 기능

| 기능 | 설명 |
|------|------|
|  파일 전송 | 클라이언트가 서버로 파일을 전송 (Java Socket 사용) |
|  파일 분석 | 서버는 전송된 파일을 실시간으로 분석 |
|  확장자 필터링 | `.exe`, `.bat`, `.dll`, `.scr`, `.js` 등 즉시 차단 |
|  키워드 기반 탐지 | `encrypt`, `ransom`, `payload`, `decryptor` 등 포함 여부 검사 |
|  인코딩 자동 감지 | UTF-8 실패 시 → MS949(CP949) 자동 재시도 |
|  로그 저장 | 악성 탐지 내역은 `log.txt` 파일로 기록 |
|  파일 삭제 | 악성 탐지 시 서버에서 자동 삭제 |
|  다중 클라이언트 | Thread 기반으로 다수 클라이언트 동시 처리 |

##  폴더 구조
```
SecureFileGuard/
├── client/
│ └── Client.java // 파일 선택 및 서버 전송
├── server/
│ └── Server.java // 서버 구동 및 클라이언트 연결
│ └── ClientHandler.java // 클라이언트 처리 및 분석
│ └── FileAnalyzer.java // 확장자 및 키워드 기반 악성 탐지
│ └── Logger.java // 서버 이벤트 및 탐지 로그 기록
└── uploads/ // 서버 수신 파일 저장 폴더
```
